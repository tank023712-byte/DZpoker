# AI Product Team Workspace

这是一个用于组织 AI 产品研发团队协作的项目骨架。

## 目录结构

```text
.
├── AGENTS.md
├── .agents/
├── docs/
├── tasks/
└── README.md
```

## 使用方式

1. 先阅读 `AGENTS.md`，确认全局规则。
2. 根据任务选择对应角色文件，例如 `.agents/product-manager.md`。
3. 将当前任务写入 `tasks/current-task.md`。
4. 将对应交付物写入 `docs/` 下的分类目录。
5. 重大变更记录到 `tasks/changelog.md`。
